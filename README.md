## Tech stack

- Pure HTML5
- Tailwind CSS (CDN)
- Google Fonts
- No build tools required — open any `.html` file directly in a browser

---

## How to edit

1. Open any `.html` file in VS Code
2. Make your changes
3. Save and push:

```bash
git add .
git commit -m "describe your change here"
git push origin main
```

Cloudflare Pages automatically redeploys within ~30 seconds.

---

## Deployment

Hosted on **Cloudflare Pages** with automatic deploys from this repository.

- Every push to `main` triggers a new deployment
- SSL certificate is automatic and free
- Custom domain connected via Cloudflare DNS

---

## Local preview

No server needed. Just open `index.html` in your browser:

```bash

start index.html

# On Mac
open index.html
```
